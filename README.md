# IMTY

Introduce Myself To You

## 개요

간편하게 포트폴리오를 만들고 관리할 수 있는 시스템

## 실행 방법

파일들을 다운로드 한 후 터미널 창에 `npm start` 입력

## 참여

**김명현**: 응원단장<br>
**김민솔**: 디자인<br>
**전준연**: 개발

# 결과

### 실패

- 기술 숙련도 부족
- 시간 부족
- 수용 인원 부족
- 등등

### 얻은 것

- **명현**: 응원단장 마스터 칭호, DB관련 지식
- **민솔**: 프로토타입 마스터 칭호
- **준연**: 각종 리액트 문법(react router, styled component 등등)
